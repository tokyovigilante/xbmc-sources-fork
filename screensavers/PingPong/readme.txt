PingPong Screensaver for XBox Media Center
By Joakim Eriksson <je@plane9.com>
The TestXBS framework and program is made by Warren

This is just a very simple screensaver that simulates a ping pong
game between two AI players.

You can set the color of the ball and the two paddles in the 
configuration file.
