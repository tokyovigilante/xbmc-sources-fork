Suggested Background Reading for Embedded WebServers

Active Server Pages

There are numerous excellent resource books on Active Server Pages. Try searching at http://www.amazon.com for "Active Server Pages".

Two recommended titles are:

  • Professional Active Server Pages, Alex Homer, et al / Paperback / Published 1997 Wrox Press Inc; ISBN: 1861000723
  • Beginning Active Server Pages 2.0, Brian Francis, et al / Mass Market Paperback Wrox Press Inc; ISBN: 1861001347
