umSetAccessLimitSecureGoAhead WebServer APIGoAhead EMF

Synopsis

Set the security setting for the URL.

Prototype

 #include "um.h"

 int umSetAccessLimitSecure(char_t * url, short secure);

Parameters

url URL name
securesecurity setting (reserved for future use)

Description

int umSetAccessLimitSecure(char_t * url, short secure);

"1" = secure
"0" = not secure

Return Value

umSetAccessLimitSecure returns 0 if successful, or -1 on error.

Example

int nCheck = umSetAccessLimitSecure(T("admin.exe"), 1);
