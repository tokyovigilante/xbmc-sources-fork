Synopsis

Retrieve the security setting for the URL.

Prototype

 #include "um.h"

 short umGetAccessLimitSecure(char_t * url);

Parameters

urlURL name

Description

short umGetAccessLimitSecure(char_t * url)

Return Value

umGetAccessLimitSecure returns the security level if successful, or -1 on error.

Example

short secure = umGetAccessLimitSecure(T("trojanhorse.cgi"));
