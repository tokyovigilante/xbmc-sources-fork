websGetRequestUserName GoAhead WebServer API GoAhead EMF

Synopsis

Return the user name specified by the browser with the current request.

Prototype

 #include "webs.h" 

 char_t *websGetRequestUserName(webs_t wp);

Parameters

wp Web server connection handle.

Description

The websGetRequestUserName procedure returns the user name supplied by the browser. After a user has entered a username and password to access a secured web page, the browser will continue to provide that username and password with every request for web documents under that point in the URL hierarchy. websGetRequestPassword and websGetRequestUsername return decoded strings.

Note: The websSecurityHandler currently ignores the user name.

Return Value

Pointer to a statically allocated string containing the decoded password. Do not call bfree.

Example

 
path = websGetRequestUserName(wp);

Stability Classification

Stable.
