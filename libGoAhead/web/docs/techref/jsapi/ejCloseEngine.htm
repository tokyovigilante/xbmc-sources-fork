ejCloseEngineGoAhead JavaScript APIGoAhead EMF

Synopsis

Close a JavaScript interpreter session.

Prototype

#include "ej.h"

void ejCloseEngine(int eid);

Parameters

eid JavaScript interpreter handle.

Description

ejCloseEngine closes a JavaScript interpreter that has been opened via ejOpenEngine.

Return Value

None.

Example

ejCloseEngine(eid);

Stability Classification

Stable.
