ejSetResultGoAhead JavaScript APIGoAhead EMF

Synopsis

Set the value of a JavaScript return variable.

Prototype

#include "ej.h"

void ejSetResult(int eid, char_t *s)

Parameters

eidJavaScript interpreter handle.
sString representing the EJ function return.

Description

The ejSetResult procedure is used for C functions which have been mapped as an ej function returning some value. By making this call, the corresponding ej function will be passed to the result. All JavaScript variables are stored as strings.

Return Value

None

EJ Example

	Color = getColor();
