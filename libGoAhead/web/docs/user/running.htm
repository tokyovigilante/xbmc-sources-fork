Accessing the GoAhead WebServer

To access the GoAhead WebServer home page:

1. Launch your browser on the same system as WebServer.

2. Launch GoAhead WebServer using web.exe (located in folder for your OS, i.e., WIN).

Note: WebServer icon should now appear in the taskbar.

3. In the browser address field enter "http://<yourhostname>" or
"http://<yourIPaddress>".

Note: defaults to port 80. Otherwise use "http://xxxxxxx:yy" where "yy" is
the correct port number.

To determine your system's host name or IP address, at the command prompt type
either "hostname" <enter> or "ipconfig" <enter> (for Win95, instead use "winipcfg" to see the IP address).

4. The server's default home page is home.asp.
