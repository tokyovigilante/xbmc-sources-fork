WebServer Getting Started Guide

The GoAhead WebServer is an open standard web server that is compliant with all necessary standards to be an effective embedded web server. GoAhead WebServer, release 2.1, supports the following platforms:

  • Windows 2000
  • Windows 98
  • Windows 95
  • Windows NT
  • Windows CE
  • VxWorks 5.3.1
  • VxWorks 5.4
  • LynxOS
  • LINUX UNIX - SCO OpenServer 3.2V5.0.0
  • eCos

The latest release of GoAhead WebServer is always available as a ZIP file for download on GoAhead's Website at http://www.goahead.com/webserver/wsregister.htm. In order to download the file you must first fill out our online registration form. Please provide complete and accurate information. After downloading the file, unzip it to a local drive on your system. Unzipping the file creates a root directory, "webserver", and sub-directories for each supported platform.

Read the GoAhead WebServer Readme (readme.txt), Release Notes (release.htm) and License (license.txt) for the latest information on the release. For aditional information on installing GoAhead WebServer, refer to the following sections of this Getting Started Guide:

  • Accessing Your GoAhead WebServer
  • Incorporation RSA Security BSAFE SSL-C
  • Managing Users
  • Troubleshooting GoAhead WebServer
