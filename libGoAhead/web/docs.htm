WebServer On-line Documents

The following on-line documents are included with WebServer 2.1:

  • Getting Started Guide
    Provides instructions on installing and using WebServer.
  • Technical Reference
    Describes each API included with this release of WebServer. These APIs are grouped according to their function: WebServer, User Management, Database, Digest Authentication, JavaScript and Basic.
  • Release Notes
    Contains current information about this release of WebServer.
  • Readme File
    Provides general information about WebServer as well as last-minute information that did not make it into the documentation.
  • License Agreement
    PLEASE CAREFULLY READ THIS AGREEMENT AS CERTIFIES THAT YOU WILL USE THE SOFTWARE ONLY IN THE MANNER PERMITTED.
  • List of Included Files
    Identifies important files included with this release of WebServer.
